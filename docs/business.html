💰 Business Model & Economics

1. Revenue Model

ArrivalX is an infrastructure company. Revenue comes from a small percentage of every transaction flowing through the settlement layer.

Revenue LineBasisRate
Agent→API SettlementTransaction volume0.50%
Agent→Merchant SettlementTransaction volume0.80-1.20%
Agent→Agent SettlementPer transaction$0.10 + 15bp
FX ConversionNotional15bp
CaaS Platform FeeMonthly$500-5,000/enterprise

Why These Rates Work

  • Agents: 0.5% on $0.50 = $0.0025. Irrelevant at machine scale.
  • Merchants: 0.8-1.2% vs. 2.9%+$0.30 for cards. 60-80% cheaper.
  • Agent-to-agent: $0.10 vs. $15-50 SWIFT. 99% cheaper.

4. Growth Model

Developer-Led Growth (B2D)

  1. Open-source SDK → 5-minute integration
  2. Sandbox free up to $100/month
  3. Framework partnerships (LangChain, AI2, Codex, MCP)
  4. x402/MPP compatibility → zero switching cost

Network Effects

  • More agents → more volume → better FX rates → more agents
  • More protocols → more use cases → more agents → more protocols
  • More jurisdictions → more counterparties → more transactions
